cr# my project

covers the basics of the git-commit workflow 

## what I practiced

- creating a repository locally 
- connecting a local repository to GitHub
- renaming branches
- checking the status of the file's
- adding files to commits
- staging and committing changes made to the file's 
- pushing code stored locally to GitHub

## cmd's used

```bash
git init
git remote add origin github-repo-link
git branch -M main
git status
git add .
git commit -m "commit message" | **commit** - a saved change. 
git push | **push** - a change that's updated to a remote repository. 
```
